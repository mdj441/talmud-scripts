## הוראות התקנה

### שלב 1 — התקנת Tampermonkey

Tampermonkey הוא תוסף חינמי להרצת סקריפטים בדפדפן. התקינו מחנות התוספים של הדפדפן שלכם:

| דפדפן | קישור |
|-------|-------|
| Chrome | [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Firefox | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/) |
| Edge | [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) |

לאחר ההתקנה יופיע אייקון של התוסף בשורת הכלים של הדפדפן.

---

### שלב 2 — הפעלת מצב מפתח ב-Chrome או Edge

> **חשוב:** החל מגרסה 5.0 של Tampermonkey, Chrome ו-Edge דורשים הפעלת מצב מפתח כדי שסקריפטים יוכלו לפעול. ב-Firefox אין צורך בשלב זה.

**ב-Chrome:**

1. פתחו `chrome://extensions/`
2. הפעילו את **"מצב מפתח"** (Developer mode) בפינה הימנית העליונה
3. אם מופיעה ב-Tampermonkey האפשרות **"Allow user scripts"** — הפעילו אותה

**ב-Edge:**

1. פתחו `edge://extensions/`
2. הפעילו את **"מצב מפתח"** (Developer mode) בתפריט הצד השמאלי

---

### שלב 3 — התקנת הסקריפט

1. לחצו על לינק ההתקנה של הסקריפט הרצוי (ראו [הסקריפטים](#הסקריפטים) למעלה)
2. Tampermonkey יזהה את הקובץ ויציג מסך התקנה
3. לחצו **Install**

#### חלופה ידנית (אם הלינק לא עובד)

1. לחצו על אייקון Tampermonkey ובחרו **"Create a new script"**
2. בחלון שנפתח: `Ctrl+A` → `Delete` → הדביקו את קוד הסקריפט → `Ctrl+S`

## שימוש

### קליטת תלמידים

1. היכנסו לאתר תלמוד ועברו לעמוד פרטי מוסד
2. הפאנל הירוק יופיע אוטומטית בפינה הימנית העליונה
3. הדביקו רשימת תלמידים — שורה לכל תלמיד בפורמט:

   | סוג | פורמט | דוגמה |
   |-----|--------|--------|
   | ת.ז. | `סניף סוג ת.ז.` | `01 300 123456789` |
   | דרכון | `סניף סוג דרכון DD/MM/YYYY ארץ` | `12 600 AB1234567 15/03/1995 USA` |

   > שורות המתחילות ב-`#` מדולגות. ניתן להפריד בטאב, פסיק או רווח.

4. הגדירו תאריך תחילת לימודים (ריק = היום)
5. לחצו **"התחל"**
6. ניתן לעצור בכל עת עם **"עצור"** או מקש `Escape`
7. בסיום — ייצאו תוצאות עם **"💾 CSV"**

ארצות נתמכות בעברית ובאנגלית: ארה"ב, אנגליה, צרפת, קנדה, אוסטרליה ועוד. ניתן גם להזין קוד ארץ ישירות (מספר).

---

### עזיבת תלמידים

1. היכנסו לאתר תלמוד ועברו לעמוד פרטי מוסד
2. הפאנל יופיע אוטומטית בפינה הימנית העליונה
3. הדביקו רשימת תלמידים — שורה לכל תלמיד בפורמט: `סניף ת.ז.`

   דוגמה:
   ```
   01 123456789
   02 987654321
   ```

4. הגדירו תאריך עזיבה וסיבה (אופציונלי)
5. לחצו **"התחל"**
6. בסיום — ייצאו תוצאות עם **"💾 CSV"**

---

### טיפול בהודעות

היכנסו לעמוד ההודעות באתר תלמוד (`Association/Default.aspx`). הפאנל הירוק יופיע אוטומטית בפינה השמאלית העליונה. בחרו אחד משני המצבים:

#### מצב גורף — לטיפול בכל ההודעות בלחיצה אחת

1. עברו ללשונית **"⚡ מצב גורף"**
2. לחצו **"▶ הפעל מצב גורף"**
3. הסקריפט יעבור על כל ההודעות במצב "נשלחה":
   - **"בקשת מעבר תלמיד"** → אישור מעבר אוטומטי
   - שאר ההודעות → סימון כטופלו

#### מצב לפי טבלה — בקרה מלאה

1. עברו ללשונית **"📋 לפי טבלה"**
2. **שלב א'** — לחצו **"🔍 סרוק"** ואז **"📥 הורד אקסל"**
3. פתחו את האקסל וערכו את עמודת **"פעולה"** לכל הודעה (אישור מעבר / דחיית מעבר / שנה מצב הודעה לטופל / דלג)
4. **שלב ב'** — בחרו את הטבלה ב-Excel (כולל כותרות), העתיקו והדביקו לתיבת הטקסט בפאנל
5. לחצו **"📥 טען"** ואז **"▶ בצע"**
6. בסיום — לחצו **"📋 העתק דוח"** לקבלת סיכום

---

### דיווח חודשי

1. היכנסו לאתר תלמוד ועברו לדף פרטי המוסד
2. לחצו על הכפתור הכחול **"▶ דיווח אוטומטי"** בפינה השמאלית התחתונה
3. הכלי יעבור סניף אחר סניף — **לא לסגור את הדפדפן בזמן הפעולה**
4. בסיום יוצג סיכום: `הסתיים! נשלחו: X | דולגו: Y`

> "דולגו" = סניפים שכבר דווחו או שלא ניתן לדווח עליהם
