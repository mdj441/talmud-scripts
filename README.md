## הסקריפטים

### [📤 דיווח חודשי אוטומטי](./talmud-monthly-report.user.js)

בלחיצה אחת — שולח דיווח חודשי לכל הסניפים אוטומטית.
הסקריפט עובר סניף אחר סניף, מדלג על סניפים שכבר דווחו, ומציג סיכום בסיום.

### [🚪 עזיבת תלמידים אוטומטית](./talmud-departure.user.js)

[**⬇️ התקנה**](../../raw/main/talmud-departure.user.js)
